# AirShare

<h1>What is it?</h1>

AirShare is a web-app to file transmit without any hardware required.\
AirShare guarantees secrecy and privacy among sent files and sender/reciever.

<h1>How to run it?</h1>

Firstly, open your terminal and run the command 

```pip install requirements.txt```


To run the file locally, simply use the command: 

```flask --app ./server.py run```

The output will give you the local link to the website opened.
