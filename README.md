# AirShare

<h1>What is it?</h1>

---

AirShare is a web-app to file transmit without any hardware required.\
AirShare guarantees secrecy and privacy among sent files and sender/reciever.

<h1>How to run it?</h1>

---

Firstly, open your terminal and venv`pip install requirements.txt`\
To run the file locally, simply use the command venv`flask --app ./server.py run`
